PIX

Jérémy (JBU), Mélanie (MEB), Brandone (BRM)

Pix s’adresse à tous les usagers français (élèves, étudiants, professionnels, décrocheurs, demandeurs d’emploi, séniors, citoyens) qui souhaitent mesurer, développer et valoriser leurs compétences numériques.

Le service se présente sous la forme d’une plateforme en ligne d’évaluation et de certification des compétences numériques.

Le but de Pix est de susciter l’envie de se former tout au long de sa vie en proposant des méthodes d’évaluation innovantes, exigeantes et bienveillantes ainsi que des recommandations de formations ciblées.

Pix est développé en Ember.js (front) et Hapi.js / Node.js (API), déployé à la fois sur OVH (API) et GitHub Pages (front).

ClubMed

Olivier (ORO), Pierrette (PBE)

Il s'agit de la refonte du site B2C, elle a été initiée il y a 3 ans suite à un cadrage 360.

Le front est disponible dans une 30aine de langues, il est maintenu par une 20aine de personnes qui commitent tous dans le même repo.

Trois features teams ont été mises en place : le shopping (présentation du produit), le BE (le tunnel de réservation), le PLE (l'espace personnel du GM).

Le site est réalisé en react, redux dans le respect des pratiques CRAFT. Nous faisons du rendu côté serveur. Nous maintenons également un server graphql pour faciliter l'intégration à l'API.

Le site est déployé sur de la barre métal chez IBM.

Europ Assistance

Swann (SKN), Anass (OUM)

Développement d’une application web pour permettre aux utilisateurs de faire une demande d’assistance lorsqu’ils tombent en panne, et cela de manière complètement digitale dans le sens où l’intégralité de la demande se fait sur l’application.

L’application est scindée en deux parties : une application React / Redux, et un serveur Node.js / Express, le tout déployé sur les services AWS (Cloudfront / Beanstalk / S3 / RDS)

Une stack react 16 à la pointe, avec du server side rendering :

  • redux : redux-saga - redux-form - react-router-redux - reselect
  • composants : recompose - material-ui - styled-component
  • testing : jest - enzyme
  • build : webpack - babel
